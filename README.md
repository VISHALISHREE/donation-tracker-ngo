# NGO Donation Tracker System

A web-based donation tracking system developed for NGOs to manage donors and donations efficiently.

## Features
- Add Donors
- Record Donations
- View Donation History
- Transparent tracking system

## Tech Stack
- PHP
- MySQL
- HTML
- CSS

## Purpose
Developed as a community-connect project to help NGOs track donations transparently.
